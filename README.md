# StartupFlow - Modern Startup Landing Page

A beautiful and modern startup website built with Next.js, Radix UI, and Tailwind CSS.

## Tech Stack

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type safety and better developer experience
- **Tailwind CSS** - Utility-first CSS framework for rapid styling
- **Radix UI** - Accessible, unstyled UI components
- **Lucide React** - Beautiful icons

## Features

- ✨ Modern, responsive design
- 🎨 Beautiful gradient backgrounds and animations
- 📱 Mobile-first responsive layout
- ♿ Accessible components using Radix UI
- 🌙 Dark mode support (system preference)
- ⚡ Optimized performance with Next.js
- 🎯 SEO-friendly structure

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the website.

## Customization

- **Brand Colors:** Update the gradient colors in the components
- **Company Name:** Change "StartupFlow" to your company name
- **Content:** Modify the hero text, features, and sections
- **Styling:** Customize Tailwind classes for your brand

## Deployment

The easiest way to deploy is using [Vercel](https://vercel.com):

```bash
npm run build
```
